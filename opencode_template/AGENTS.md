# OpenCode Personal Template - Agent Roles & Rules

- Planner: strategic task decomposition and planning
- Builder: code writer/template filler
- Architect: high-level design decisions for template structure
- Reviewer: code/documentation quality checks
- Tester: test strategy and validation
- Debugger: issue resolution and troubleshooting
- Docs: documentation steward

- Rules: Always separate system-fixed vs editable content. Bootstrap via setup_template.sh/.ps1. Update TASK.md after interactions. Use PROGRESS.md for progress logging. Use HANDOFF.md for milestones.
